### Expo managed workflow
- MAC
    - ( Install brew )
    - brew --version
    - brew help
    - brew info
    - brew list
    - Path:
        - Node.js v18.12.1 to /usr/local/bin/node
        - npm v8.19.2 to /usr/local/bin/npm
    - npx create-expo-app my-app
    - cd my-app
    - npx expo start
